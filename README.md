## 🚀 Running Locally

**1. Clone the repo**
```bash
git clone https://github.com/pragyamittal0/my-portfolio.git
cd my-portfolio
````

**2. Install dependencies**

```bash
npm install
```

**3. Start the app**

```bash
npm start
```

Then visit `http://localhost:3000`.
